### 📄 مسار الملف الافتراضي: docs/javascript/edit-attention.md

#### 1. الوظيفة الأساسية الفعالة (Core Purpose):
محرر أوزان الانتباه في البرومبت (128 سطر). Ctrl/Cmd + ArrowUp/Down يُعدّل صيغة `(text:weight)`.

التفاصيل: Delta: 0.1 لـ `()`، 0.05 لـ `<>`. يلف النص المحدد تلقائياً في `(text:1.0)` إذا لم يكن موزوناً.

#### 2. الكلاسات والدوال الرئيسية (Classes & Functions):
- معالجة أحداث Ctrl/Cmd + ArrowUp/Down.
- تحليل صيغة الأقواسsupports both `()` and `<>`.
- تحديد الأقواس المقابلة أو الكلمة بالكامل.

#### 3. خريطة الاعتماديات والعلاقات (File Dependencies):
- **يعتمد على**: DOM Selection API
- **يُستخدم بواسطة**: webui.py
- **الأثر البرمجي**: تغيير هنا يؤثر على كيفية تعديل المستخدم لأوزان البرومبت
